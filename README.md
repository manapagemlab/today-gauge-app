# 今日のしんどさゲージチェック

親子でその日の疲れ・空腹・不安・刺激などを見える化する、スマホ向けWebアプリです。

このアプリは医療的な診断ではありません。今日の状態を親子で整理するための目安として使います。

## 機能

- 8つのゲージを各3問、合計24問でチェック
- 回答と補足メモからゲージスコアを計算
- 上位3項目と今日のひとことを表示
- 今日できる作戦アドバイスを表示
- 結果カードをPNG画像として保存

## 起動

```bash
npm install
npm run dev
```

ブラウザで `http://localhost:3000` を開きます。

## テスト

```bash
npm test
npm run build
```
